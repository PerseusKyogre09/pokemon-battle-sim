# Pokémon Battle Simulator

A feature-rich Pokémon battle simulator built with FastAPI, React (Next.js), and Supabase. This web application allows players to engage in strategic turn-based battles with authentic Pokémon mechanics.

## 🎮 Features
- **Authentic Pokémon Battles**: Experience turn-based battles with official Pokémon mechanics.
- **Strategic Movesets**: Pokémon come with optimized movesets based on their type and abilities.
- **Type Advantages**: Full implementation of Pokémon type effectiveness.
- **Modern UI**: Built with Next.js and Tailwind CSS for a premium look.
- **Supabase Integration**: Securely serves audio assets and battle data.

## 🚀 Deployment
- **Frontend**: Deployed on Vercel.
- **Backend**: Deployed on Hugging Face Spaces (Docker).
